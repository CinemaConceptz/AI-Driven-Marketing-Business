EPK Generation Logic
Verified Sound – Representation Platform

Phase-1 – EPK Engine Design

Objective

Design a structured EPK generation engine that transforms onboarding data into a standardized professional representation profile.

The system must ensure:

• Consistent formatting
• Professional tone
• Strategic alignment
• Label-ready positioning
• Scalability across genres

Core Output Components

The Verified Sound EPK includes:

Full Professional Biography

Short Press Bio (200–300 words)

Strategic Positioning Summary

Label Pitch Paragraph

Genre Alignment Classification

Release Readiness Overview

Market Target Summary

Representation Snapshot

Input → Output Mapping
From Identity & Positioning Section:

Used to generate:

• Opening biography paragraph
• Career narrative framing
• Project uniqueness angle

From Musical Style & Influences:

Used to generate:

• Genre description language
• Comparative positioning references
• Catalog alignment summary

From Catalog Assessment:

Used to generate:

• Release readiness score
• Professional credibility section
• Track record overview

From Market Goals:

Used to generate:

• Target market paragraph
• Long-term positioning summary
• Strategic outlook section

Output Structure Format
Section 1 – Artist Overview

Professional biography (600–900 words)

Tone:
Industry-standard
Press-ready
Confident
No exaggeration

Section 2 – Short Press Bio

Condensed version for email pitching.

Section 3 – Strategic Positioning Summary

Clear explanation of:

• Where the artist fits in current market
• Label type alignment
• Genre compatibility

This section feeds compatibility scoring.

Section 4 – Label Pitch Paragraph

One paragraph designed specifically for demo outreach emails.

This must be:

Concise
Clear
Strategic
Not generic

Section 5 – Representation Snapshot

Bullet-style summary including:

• Primary Genre
• Secondary Genres
• Years Active
• Release Count
• Target Market
• Contract Status
• Recommended Representation Tier

Governance Rules

The EPK engine must:

• Avoid exaggerated claims
• Avoid unverifiable statistics
• Avoid false industry references
• Maintain professional tone

If onboarding data is weak, the output must remain honest and structured.

Upgrade Capability

Future phases may include:

• Dynamic web-based EPK hosting
• Downloadable PDF export
• Label-specific formatting versions
• Sync-focused version
• Press-focused version

Core Principle

The Verified Sound EPK is not a resume.

It is a structured representation document designed for strategic label targeting.
